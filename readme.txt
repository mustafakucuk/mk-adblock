If your visitors uses Adblock, hide your elements, videos and show alert block!

== Description ==
1. Hide Elements ( optional ) - You can hide multiple html elements.
2. Hide Videos ( optional ) - You can hide videos.
3. Required ( optional ) - You can make required or your visitors can close alert and see content.

Did you see? Everything you want!

Thanks!

== Installation ==
Please follow these steps to install the plugin.

1. Upload `mk-adblock` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `MK Adblock` menu in WordPress
